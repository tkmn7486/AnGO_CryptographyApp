# AnGO_CryptographyApp

このアプリについて＞
日本語のひらがなを暗号にすることができます。
その逆、暗号をひらがなに復元することもできます。

CSVファイルを辞書型に直す処理を組み込んでいるため、ひらがな以外の言語（アルファベットや中国語など）に対応させるのも容易です。

アップデートしたい機能＞
・複数タイプの暗号化処理を行えるようにする
　→ タイプ１～５をランダムに行い、行った処理の番号を出力画面に表示する、など

・CSV　アルファベットを対応させる
　→ アルファベットはひらがなを対応させればOK？
 
・TwitterやLINEに出力する機能を付けてみる

・IOSやAndroidに対応させる
　→ WEBアプリという手もある。
 　 そうでなければkivyなど他のフレームワークで作り直してみる
